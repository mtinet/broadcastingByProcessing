# broadcastingByProcessing
# 프로세싱을 이용하여 브로드캐스팅하기  

### 1. 서버는 랜덤으로 수치를 올리면서 화면에 뿌림  
### 2. 클라이언트가 접속하면 해당 클라이언트의 ip를 콘솔에 띄움  
### 2. 랜덤으로 만들어진 해당 수치를 클라이언트에 보냄  
### 3. 클라이언트는 해당 수치를 받아 화면에 색깔로 뿌림
